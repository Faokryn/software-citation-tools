Getting Involved

Software Citation Tools is being developed as an open source project. As such, participation and contribution is encouraged. Please check out our Contributing Guidelines on GitHub before contributing.

Join the Conversation

Software Citation Tools has plenty of ways to get involved, even if you can't contribute code! We have several discussion issues on GitHub Issues were we pose questions to help shape the direction and requirements of the project. Feel free to participate! You can also open an issue if you discover a bug or would like to request a feature.

Contribute to the Project

As an open source project, code contributions are encouraged! We record bugs and planned features in GitHub Issues. Please note that the issues for all of the tools are kept on the Issues page of the main repository. Check out our Contributing Guidelines for more details.

Contact

Feel free to reach out to the team at any time with any questions or comments you may have using the team's Gitter room.
