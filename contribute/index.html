Contributing

The Software Citation Tools are being developed as open source software, following open source best practices. Before contributing, it is a good idea to be familiar with the Software Citation Principles being used. All development is being done via the GitHub Repository. Please see the repo for more information on how you can contribute (coming soon).

A public chatroom and information on communicating with the development team are forthcoming.
