Contributing

The Software Citation Tools are being developed as open source software, following open source best practices. Before contributing, it is a good idea to be familiar with the Software Citation Principles being used. All development is being done via the GitHub Repository. Please see the contributing doc for more information on how you can contribute.

A public chatroom is available on Gitter and mirrored on the #software-citation-tools IRC channel on Freenode. Feel free to join if you have a question for the development team, or just want to participate in the discussion of the direction of the project.
